# Mickey Speedster

**Mickey Speedster** é um jogo de plataforma 2D side-scrolling focado em velocidade e momentum, adaptando as mecânicas clássicas de "loopings" e física de inclinação para o personagem Mickey Mouse.

## Visão Geral
O objetivo é adaptar as mecânicas clássicas de jogos de velocidade, utilizando a engine AntiGravity.

## Stack Técnica
- **Engine**: AntiGravity (JS/TS)
- **Versionamento**: Git (GitHub)
- **Interface de Git**: GitHub Desktop
- **Hospedagem/Deploy**: GitHub Pages
- **Assets**: Spritesheets 2D (estilo 16-bit ou moderno)

## Estrutura de Pastas
- `/assets`: Sprites, sons e mapas (Tiled)
- `/src`: Código fonte do jogo (AntiGravity scripts)
- `index.html`: Ponto de entrada do jogo

## Mecânicas Principais (Mickey vs Sonic)
- **Spin Dash**: Mickey pode usar as orelhas para girar ou usar um "dash" clássico.
- **Coletáveis**: Estrelas Mágicas ou Luvas Brancas substituem as clássicas argolas.
- **Física e Momentum**: Cálculo de aceleração constante do AntiGravity para simular o ganho de velocidade em descidas e sistema de sensores de colisão para permitir subir paredes e tetos em alta velocidade.
